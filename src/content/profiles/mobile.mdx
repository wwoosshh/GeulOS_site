# 모바일 OS

오늘날 모바일은 **Android·iOS 양강**이지만, 그 자리에 오기까지 여러 OS가 등장했다 사라졌다.

## Android — Linux 기반 1위

[Android](https://en.wikipedia.org/wiki/Android_(operating_system))는 **Google**과 Open
Handset Alliance가 개발한, *수정된 Linux 커널* 기반 OS다. 첫 상용 기기 **HTC Dream이
2008년 9월 23일** 출시됐다([Android, Wikipedia](https://en.wikipedia.org/wiki/Android_(operating_system))).
오픈소스 부분(AOSP)은 **Apache 2.0**, 커널 수정분은 GPLv2. 월 30억+ 활성 사용자를 가진
*세계에서 가장 널리 쓰이는 OS*다.

## iOS — Apple의 모바일 OS

[iOS](https://en.wikipedia.org/wiki/IOS)는 **2007년 6월 29일** 초대 iPhone과 함께 나왔다.
처음엔 "iPhone OS"였다가 **2010년 6월 iOS로 개명**됐고, **2019년 iPadOS**가 분리됐다
([iOS, Wikipedia](https://en.wikipedia.org/wiki/IOS)). [macOS](/profiles/apple)와 같은
Mach 마이크로커널 + FreeBSD 기반의 Unix-like 시스템이다.

## HarmonyOS — 분산 OS의 도전 (2019~)

[HarmonyOS](https://en.wikipedia.org/wiki/HarmonyOS)는 **Huawei**가 **2019년 8월 9일** 공개한
*분산 OS*로, 스마트폰·태블릿·TV·워치·PC를 아우른다. 버전 1~4는 Android 앱 호환(AOSP 통합)을
지원했지만, **HarmonyOS 5(구 HarmonyOS NEXT)부터 Android 코드를 완전히 제거**하고 자체 앱
포맷만 지원한다([HarmonyOS, Wikipedia](https://en.wikipedia.org/wiki/HarmonyOS)).

## 커널 관점

거의 모든 현대 모바일 OS가 [Unix 계열](/profiles/unix-bsd)이다 — Android는 Linux,
iOS는 Darwin/BSD. 모바일의 시장·점유율 맥락은 [용도별 분류](/classification/usage#모바일-os)에서
다룬다.
