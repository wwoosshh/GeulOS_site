# 주요 OS 심화 — 계보 지도

[분류 5축](/classification)이 *가로로* OS를 나눈다면, 이 섹션은 대표 OS 가문을 *세로로*
파고든다 — 언제 누가 만들었고, 무엇에서 갈라져 나왔으며, 무엇이 기술적으로 다른가.
모든 연도·계보는 출처 링크로 검증한다.

## 두 거대 가문

현대 범용 OS 대부분은 두 뿌리로 수렴한다.

- **Unix 계열** — 1969년 벨 연구소에서 시작([Unix, Wikipedia](https://en.wikipedia.org/wiki/Unix)).
  여기서 BSD → macOS/iOS, System V → 상용 Unix, 그리고 Unix를 본떠 새로 쓴 **Linux**가
  갈라진다. 오늘날 서버·슈퍼컴·모바일(Android/iOS)·콘솔이 사실상 모두 Unix 계열이다.
- **Windows 계열** — 1981년 MS-DOS에서 출발해 1985년 GUI 셸 Windows, 1993년 새로 설계한
  NT 라인으로 이어졌고, 2001년 Windows XP에서 소비자(9x)·기업(NT) 라인이 통합됐다
  ([Microsoft Windows, Wikipedia](https://en.wikipedia.org/wiki/Microsoft_Windows)).
