# Windows — DOS에서 NT까지

데스크탑의 표준. 그러나 오늘의 Windows는 *두 개의 다른 OS 라인*이 합쳐진 결과다.

## 출발 — MS-DOS와 GUI 셸 (1981~)

[Windows](https://en.wikipedia.org/wiki/Microsoft_Windows)의 뿌리는 1981년 IBM PC와 함께
나온 명령줄 OS **MS-DOS**다. 그 위에 GUI를 올린 **Windows 1.0이 1985년 11월 20일** 출시됐다
([Microsoft Windows, Wikipedia](https://en.wikipedia.org/wiki/Microsoft_Windows)).
초기 Windows 1.0~3.x와 이후 **Windows 95(1995년 8월 24일)**·98·Me는 모두 *DOS 기반의
9x 계열*로, 32비트 지원과 시작 메뉴를 도입했지만 안정성·보안에 한계가 있었다.

## NT 라인 — 새로 설계한 커널 (1993~)

Microsoft는 DOS와 무관하게 **Windows NT 3.1을 1993년 7월 27일** 별도 라인으로 출시했다.
NT는 [하이브리드 커널](/classification/kernel) 기반의 *보안·멀티유저·32비트* 설계로,
서버와 기업 환경을 겨냥했다([Microsoft Windows, Wikipedia](https://en.wikipedia.org/wiki/Microsoft_Windows)).

## 두 라인의 통합 — Windows XP (2001)

**Windows XP(2001년 10월 25일)**가 소비자용 9x 라인과 NT 아키텍처를 *하나로 통합*했다.
이후 모든 Windows는 NT 커널 위에서 동작한다.

| 버전 | 출시 | 라인 |
|---|---|---|
| Windows 1.0 | 1985-11-20 | DOS GUI 셸 |
| Windows 95 | 1995-08-24 | 9x (소비자) |
| Windows NT 3.1 | 1993-07-27 | NT (기업·서버) |
| **Windows XP** | 2001-10-25 | **9x + NT 통합** |
| Windows Vista | 2007-01-30 | NT |
| Windows 7 | 2009-10-22 | NT |
| Windows 10 | 2015-07-29 | NT |
| Windows 11 | 2021-10-05 | NT |

> 연도는 모두 [Microsoft Windows, Wikipedia](https://en.wikipedia.org/wiki/Microsoft_Windows)에서 확인.

## 오늘의 위치

데스크탑 점유율 1위이며, NT 커널은 데스크탑뿐 아니라 **Xbox**(Windows NT + Hyper-V),
Windows Server 등으로 확장됐다. 데스크탑·서버·게임에서의 위치는
[용도별 분류](/classification/usage)와 [응용 분야별](/classification/application)에서 다룬다.
