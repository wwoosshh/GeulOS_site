# 보고서 정보

## 과제 정보

운영체제 강의 과제 — *운영체제를 시대·용도·응용 등에 따라 분류하고 그 종류들을 조사·설명*.

본 자료실은 위 과제를 *위키 형식 웹사이트*로 제출한 결과물. 객관 자료(Wikipedia, 산업 통계, 공식 문서) 기반 분류 5축으로 종합 정리 + 신생 분류 사례로 GeulOS 프로젝트 소개.

## 작성

- **작성일**: 2026-05-31
- **언어**: 한국어
- **출처 검증**: 모든 통계 수치는 [참고자료](/references) 섹션에 URL 명시.

## 사이트 정보

- **기술 스택**: Next.js 15 + Nextra 4 (Pages Router 호환 App Router) + React 19.
- **배포**: Vercel.
- **소스**: GeulOS GitHub repository 일부로 관리. 콘텐츠는 MDX (Markdown + React 컴포넌트).
- **타이포그래피**: Pretendard (본문) + JetBrains Mono (코드).
- **색감**: 인디고/슬레이트 베이스 (AI 클리셰 회피 — 보라/핑크 그라데이션 X).

## 디자인 원칙

본 사이트는 **정보 밀도 우선**으로 설계됐다. 위키피디아·MDN Web Docs·Stripe Docs와 같은 *고전적 문서 사이트* 미학:

- 카드 그리드는 *홈 hero 진입점*에만 한정.
- 본문은 *문단 + 표 + 코드 블록*. 모든 정보를 카드로 감싸지 않음.
- 색은 차분한 1색 액센트. 그라데이션·글로우·backdrop blur 등 *AI 풍 시각 트릭* 회피.
- 좌측 사이드바 + 우측 TOC가 기본 — 1-hop 도달.
