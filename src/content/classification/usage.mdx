# 용도별 분류

대상 환경/시스템에 따라 OS가 갈린다. 같은 Linux 커널도 *데스크탑·서버·임베디드·슈퍼컴*
용도에 따라 배포판과 설정이 달라진다.

## 데스크탑·랩탑 OS

일반 사용자용 GUI 환경. 사무·게임·미디어·개발.

| OS | 개발사 | 커널 | 라이선스 | 비고 |
|---|---|---|---|---|
| **Windows 10/11** | Microsoft | NT (hybrid) | 상용 | 데스크탑 점유율 ~70% (StatCounter 2025) |
| **macOS** | Apple | XNU (hybrid, Mach+BSD) | 상용 | Apple Silicon 전환 |
| **Ubuntu** | Canonical | Linux (monolithic) | GPL/오픈 | 가장 대중적 Linux |
| **Fedora** | Red Hat 후원 | Linux | GPL | RHEL 업스트림 |
| **Arch Linux** | 커뮤니티 | Linux | GPL | 롤링 릴리스, 미니멀 |
| **Debian** | 커뮤니티 | Linux | GPL/오픈 | **1993년 Ian Murdock** 창립, Ubuntu·Mint 등 수많은 파생의 *모태* ([Debian](https://en.wikipedia.org/wiki/Debian)) |
| **Linux Mint** | 커뮤니티 | Linux | GPL | Ubuntu 기반, Windows 사용자 친화 |
| **Pop!_OS** | System76 | Linux | GPL/오픈 | **2017년** System76이 Ubuntu 기반으로 개발, 개발자·타일링 친화 ([Pop!_OS](https://en.wikipedia.org/wiki/Pop!_OS)) |
| **openSUSE** | SUSE 후원 | Linux | GPL | SUSE 업스트림, YaST 관리도구 ([openSUSE](https://en.wikipedia.org/wiki/OpenSUSE)) |
| **ChromeOS** | Google | Linux | 상용+OSS | 웹 앱 중심, 교육 시장 강세 |

> 위 배포판 대부분이 **Debian 계열**(Ubuntu·Mint·Pop!_OS·elementary OS)과 **Red Hat
> 계열**(Fedora·RHEL), **SUSE 계열**(openSUSE)로 묶인다. 같은 [Linux 커널](/profiles/linux)
> 위에서 데스크탑 환경·패키지 관리자만 다르다.

## 서버 OS

대량 요청 처리·고가용성·원격 관리.

- **Linux가 서버 시장 약 51%** (2026 기준, 일부 자료는 63%까지). 웹 서버 점유율은 **96.3%**가 Linux 기반.
- **Linux 배포판 (엔터프라이즈)** 점유:
  - **Red Hat Enterprise Linux (RHEL)** — 43.1%
  - **Ubuntu Server** — 33.9%
  - **Debian** — 16%
  - **SUSE Linux Enterprise** — 11.2%
- **Windows Server** — Active Directory·Exchange·.NET 기반 엔터프라이즈.
- **FreeBSD** — Netflix CDN, 일부 ISP.
- **Solaris (Oracle)** — 레거시 엔터프라이즈.
- **IBM AIX** — IBM Power 시스템.

## 임베디드 OS

가전·산업 장비·IoT 기기. 적은 메모리/저전력.

- **임베디드 Linux** — 라우터·NAS·셋톱박스 등 광범위.
- **VxWorks** (Wind River) — 1,000+ 제품, 1억+ 기기 탑재. 우주왕복선·화성 탐사선.
- **QNX Neutrino** (BlackBerry) — 마이크로커널. 차량 6천만 대+ 탑재.
- **FreeRTOS** (Amazon) — 가장 인기 있는 오픈 RTOS. 매 170초마다 다운로드.
- **Zephyr** (Linux Foundation) — 오픈, 스케일러블 RTOS. 웨어러블·스마트홈.
- **ThreadX / Azure RTOS** (Microsoft) — 60억+ 디바이스. 2026 안전 중심 RTOS 1위.

## 실시간 OS (RTOS)

작업 데드라인을 *예측 가능한 시간 안에* 보장. 자동차·항공·산업 제어·의료.

- **Hard RTOS**: 데드라인 미수성 시 시스템 실패 — 항공기 제어, ABS 브레이크.
- **Soft RTOS**: 미수성 시 품질 저하만 — 멀티미디어 스트리밍.

대표:

- **VxWorks** — 결정론적·저지연·항공/방산 인증.
- **QNX** — 마이크로커널, ISO 26262 ASIL-D 차량 인증.
- **INTEGRITY** (Green Hills) — 의료·방산.
- **RTLinux / Xenomai** — Linux 위 실시간 패치.
- **FreeRTOS / Zephyr** — 마이크로컨트롤러용.

## 메인프레임 OS

대형 기업·금융·정부의 핵심 트랜잭션 시스템.

- **IBM z/OS** — IBM Z 시리즈 (z14/z15/z16). 은행·항공권 예약.
- **z/VM** — 가상화 메인프레임.
- **z/TPF** (Transaction Processing Facility) — 초당 수만 트랜잭션.
- **Unisys MCP / OS 2200** — Unisys ClearPath.

## 슈퍼컴퓨터 OS

과학 계산·기후 시뮬레이션·AI 학습.

- **TOP500 슈퍼컴 100%가 Linux** (2017년 11월 이후 8년 연속, 2026 May 기준).
- 1위 **El Capitan** (미국 LLNL, 1.742 ExaFLOPS) — RHEL 기반.
- HPE Cray 시스템은 **Cray Linux Environment** 사용.

## 분산 OS

여러 컴퓨터를 단일 시스템처럼 추상화. 학술 비중 큼.

- **Plan 9 from Bell Labs** (1992) — UNIX 후계 연구. *모든 것이 파일*.
- **Inferno** (Lucent, 1996) — Plan 9 진화형. 가상 머신(Dis) 위.
- **Amoeba** (Tanenbaum, 1981) — Multi-CPU 추상화.
