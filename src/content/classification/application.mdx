# 응용 분야별 분류

같은 임베디드/실시간 OS도 *어떤 산업/기기에 들어가느냐*에 따라 시장 동향과 요구사항이
크게 달라진다. 자동차·IoT·게임 콘솔이 대표적인 큰 시장이다.

## 자동차 OS

2024년 자동차 OS 시장 약 134억 달러, 2033년 460억 달러 (CAGR ~15%) 예상.

- **Android Automotive OS (AAOS)** — 2026년 *신차 중급~고급 모델 90%* 사용 (Google). HVAC·내비게이션 통합.
- **QNX** — 안전 임계 35~38% 점유. ADAS·계기판·제어. ISO 26262 ASIL-D 인증.
- **Automotive Grade Linux (AGL)** — Linux Foundation 후원. 13.7% → 2026 23% 예상. Toyota·Mazda 등.
- **AUTOSAR Adaptive** — 차량 ECU 표준 미들웨어.

> 현대 차량은 **단일 OS가 아닌 멀티 OS** — 안전(QNX) + 인포테인먼트(Android/Linux) +
> 자율주행(QNX/Linux) 도메인별 분리.

## 사물인터넷 (IoT)

저전력·저메모리(KB~MB)·무선 통신.

- **Mbed OS** (Arm) — Cortex-M 마이크로컨트롤러. 클라우드 연결 API.
- **Contiki-NG** — 저전력 무선(6LoWPAN), IPv6, MQTT.
- **RIOT** — 오픈 커뮤니티, 8/16/32비트 폭넓게 지원.
- **TinyOS** — 센서 네트워크 연구용 효시.
- **Tizen** (삼성) — TV·웨어러블·가전.
- **HarmonyOS** — 멀티 디바이스 분산.

## TV · 셋톱박스 OS

스마트 TV·스트리밍 기기 시장도 OS 경쟁이 치열하다. 모바일과 달리 *원격 리모컨 UI +
스트리밍 앱 생태계*가 핵심이다.

| OS | 개발 | 기반 | 비고 |
|---|---|---|---|
| **webOS** | LG (원래 Palm) | Linux | Palm이 2009년 모바일용으로 출시, HP·LG 인수 후 **2014년부터 LG 스마트 TV 플랫폼** ([webOS](https://en.wikipedia.org/wiki/WebOS)) |
| **Tizen** | 삼성·Linux Foundation | Linux | 삼성 스마트 TV·가전의 기본 OS ([Tizen](https://en.wikipedia.org/wiki/Tizen)) |
| **Android TV / Google TV** | Google | Android | 2014년 Google TV(2010~)의 후속으로 출시, **2020년 'Google TV' UI 도입** ([Android TV](https://en.wikipedia.org/wiki/Android_TV)) |
| **tvOS** | Apple | iOS 기반 | **2015년 10월 29일** Apple TV용으로 출시(개명) ([tvOS](https://en.wikipedia.org/wiki/TvOS)) |
| **Fire OS** | Amazon | Android(AOSP) 포크 | Fire TV·Fire 태블릿용. Google 서비스 미포함 ([Fire OS](https://en.wikipedia.org/wiki/Fire_OS)) |
| **Roku OS** | Roku | Linux | **2004년**부터 스트리밍 플레이어, 2014년 스마트 TV로 확장 ([Roku OS](https://en.wikipedia.org/wiki/Roku_OS)) |

> TV OS 대부분이 [Linux](/profiles/linux) 또는 [Android](/profiles/mobile)(=Linux 커널)
> 계열이고, Apple만 [iOS 계열](/profiles/apple)이다.

## 웨어러블 OS

스마트워치·밴드. 초저전력 + 작은 화면 + 센서 통합이 관건이며, 모바일 OS의 *축소·파생*
형태가 많다.

| OS | 개발 | 기반 | 비고 |
|---|---|---|---|
| **watchOS** | Apple | iOS 기반 | **2015년 4월 24일** Apple Watch용 출시 ([watchOS](https://en.wikipedia.org/wiki/WatchOS)) |
| **Wear OS** | Google | Android | 원래 *Android Wear*로 **2014년 3월** 발표, **2018년 Wear OS로 개명** ([Wear OS](https://en.wikipedia.org/wiki/Wear_OS)) |
| **Tizen** | 삼성 | Linux | 삼성 갤럭시 워치(초기) ([Tizen](https://en.wikipedia.org/wiki/Tizen)) |

## 항공·우주·방산 (Mission-Critical)

- **VxWorks** — NASA 화성 탐사선 (Spirit, Opportunity, Perseverance), 보잉 787.
- **INTEGRITY-178B** — 미 공군 F-35 전투기.
- **RTEMS** (Real-Time Executive for Multiprocessor Systems) — NASA 위성.

## 의료

- **VxWorks / QNX** — MRI·CT 스캐너·인공호흡기·수술 로봇 (FDA Class III 인증).
