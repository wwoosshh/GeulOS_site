# 요약 표 — 대표 OS 다섯 축 정리

12개 대표 OS를 다섯 축에 한눈에 매핑한다.

| OS | 시대 | 용도 | 구조 | 라이선스 | 핵심 강점 |
|---|---|---|---|---|---|
| Linux | 5세대~ | 서버/슈퍼컴/데스크탑/모바일/IoT | 모놀리식 | GPL | 압도적 점유율, 모든 영역 |
| Windows NT | 5세대~ | 데스크탑/서버/게임 | 하이브리드 | 상용 | 데스크탑 표준, 생태계 |
| macOS | 5세대~ | 데스크탑/크리에이티브 | 하이브리드 (XNU) | 상용 | 통합 HW/SW |
| Android | 6세대 | 모바일/자동차/TV | Linux 모놀리식 | Apache | 모바일 1위 |
| iOS | 6세대 | 모바일 | XNU 하이브리드 | 상용 | 통합 생태계 |
| FreeBSD | 5세대 | 서버/콘솔/네트워크 | 모놀리식 | BSD | Netflix, PS5 |
| QNX | 4세대~ | 자동차/의료/방산 | 마이크로커널 | 상용 | 안전 인증 |
| VxWorks | 4세대~ | 항공/방산/산업 | 모놀리식 | 상용 | 결정론적 RTOS |
| FreeRTOS | 5세대 | IoT/MCU | 모놀리식 (μ) | MIT | 오픈 RTOS 1위 |
| z/OS | 3세대~ | 메인프레임 | 모놀리식 | 상용 | 금융 트랜잭션 |
| Horizon | 6세대 | Switch 콘솔 | 커스텀 (FreeBSD+Android) | 상용 | 휴대/거치 모드 |
| seL4 | 6세대 | 방산/연구 | 마이크로커널 | GPL | *수학적 형식 검증* |

## 결론 — 다양성의 의미

운영체제는 **단일 정답이 없는 영역**이다. 슈퍼컴에서는 Linux가, 메인프레임에서는
z/OS가, 차량에서는 QNX와 Android Automotive가, 마이크로컨트롤러에서는 FreeRTOS가
*각자의 제약*에 가장 잘 맞기 때문에 살아남았다.

분류 축마다 **다른 그림**이 보인다:

- **시대별** — PC → 모바일 → 클라우드/AI로의 패러다임 전환.
- **용도별** — 데스크탑은 Windows·macOS·Linux, 서버는 Linux 일변도, 모바일은 Android·iOS 양강.
- **구조별** — 성능 우선의 모놀리식 vs 안전 우선의 마이크로커널 사이의 *영원한 트레이드오프*.
- **응용 분야별** — *안전 인증·시간 보장*이 결정적인 자동차·항공 영역과 *확장성·생산성*이 우선인 일반 영역의 분기.

상호작용하려면, GUI를 픽셀로 모방하는 대신 의미(semantic) 차원에서 동작할 통로가
