# 시대별 분류

운영체제 발전사는 통상 5~6 세대로 나눈다. 각 세대는 *하드웨어 패러다임 전환*과
짝지어 등장했다.

## 제1세대 (1940~1955) — OS 없음

- 진공관 시대의 컴퓨터(ENIAC, UNIVAC I 등).
- **운영체제가 존재하지 않음.** 사용자가 직접 토글 스위치·천공 카드·자기 테이프로 프로그램을 일일이 로드.
- 모든 프로그램이 하드웨어 전체 사양을 직접 알아야 동작.

## 제2세대 (1955~1965) — 일괄 처리 (Batch Processing)

트랜지스터 시대. 운영자가 프로그램을 배치(batch)로 묶어 처리.

| OS | 출시 | 비고 |
|---|---|---|
| **GM-NAA I/O** | 1956 | *최초의 실제 사용된 운영체제*. General Motors–North American Aviation, IBM 704용 |
| **IBSYS** | 1960년대 | IBM 7090/7094용 기본 OS |

- 한 번에 한 프로그램만 실행. CPU 유휴 시간 큼.

## 제3세대 (1965~1980) — 다중 프로그래밍·시분할

집적회로(IC) 도입. 여러 프로그램을 동시에 메모리에 두고 CPU를 번갈아 할당.

| OS | 출시 | 특징 |
|---|---|---|
| **OS/360** | 1966 (IBM) | 동일 OS가 다양한 System/360 모델에서 동작. *호환성*과 *멀티프로그래밍* 도입 |
| **CTSS / Multics** | 1961/1965 (MIT) | *시분할* 시스템. 여러 사용자 단말기 동시 접속. 보호 링(ring) 개념 |
| **UNIX** | 1969 (AT&T Bell Labs) | Ken Thompson·Dennis Ritchie. Multics 단순화 + C로 재작성해 이식성 확보 |
| **TOPS-10 / TOPS-20** | 1970년대 (DEC) | PDP-10 시분할 |
| **VAX/VMS (→ OpenVMS)** | 1977 발표 (DEC) | VAX 미니컴퓨터용. *클러스터링* 기반 고가용성으로 은행·의료·통신에서 지금도 사용 ([OpenVMS](https://en.wikipedia.org/wiki/OpenVMS)) |

이 시기에 [가상 메모리](/concepts/memory)·계층적 [파일 시스템](/concepts/filesystem) 등
현대 OS의 기초가 자리잡았다. VMS는 DEC가 **1977년 10월 발표**하고 1978년 V1.0을 낸 미니컴퓨터
OS로, 17년 무중단 클러스터 운영 사례가 보고될 만큼 고가용성에 특화됐다
([OpenVMS, Wikipedia](https://en.wikipedia.org/wiki/OpenVMS)).

## 제4세대 (1980~1995) — 개인용 컴퓨터·GUI

마이크로프로세서(Intel 8086, Motorola 68000) 확산으로 *한 사람이 한 컴퓨터*를 갖는 시대.

| OS | 출시 | 특징 |
|---|---|---|
| **CP/M-80** | 1974 (Digital Research) | 8비트 PC의 표준 |
| **MS-DOS / PC DOS** | 1981 (Microsoft·IBM) | IBM PC 출시. 명령줄 기반 |
| **Classic Mac OS** | 1984 (Apple) | *상용 GUI*의 대중화 |
| **AmigaOS** | 1985 (Commodore) | 선점형 멀티태스킹 + 멀티미디어 강점 |
| **Atari TOS** | 1985 | Atari ST용 |
| **Windows 1.0~3.x** | 1985~1992 (Microsoft) | DOS 위 GUI 쉘 ([Windows](/profiles/windows)) |
| **OS/2** | 1987 (IBM·Microsoft 공동) | 32비트 멀티태스킹. 1990년 결별 후 IBM 단독, **Warp 3(1994)** ([OS/2](https://en.wikipedia.org/wiki/OS/2)) |
| **NeXTSTEP** | 1989 (NeXT) | Steve Jobs의 NeXT가 개발. 훗날 [macOS·iOS의 토대](/profiles/apple) ([macOS](https://en.wikipedia.org/wiki/MacOS)) |

## 제5세대 (1995~2007) — 네트워크·분산·서버

인터넷 보편화. 32비트→64비트 전환. 클라이언트-서버 모델.

| OS | 출시 | 특징 |
|---|---|---|
| **Windows NT 계열** | 1993 (Microsoft) | NT 3.1 → 2000 → XP → Vista → 7 → ... 모놀리식 DOS 기반 9x 라인 흡수 |
| **Linux** | 1991 (Linus Torvalds) | 오픈소스 Unix-like 커널. 서버·임베디드·슈퍼컴 표준 |
| **FreeBSD / OpenBSD / NetBSD** | 1993~ | BSD Unix 후예 |
| **macOS X** | 2001 (Apple) | Darwin(BSD+Mach) 커널 기반 ([Apple 계보](/profiles/apple)) |
| **Solaris** | 1992~ (Sun Microsystems) | 엔터프라이즈 Unix |
| **BeOS** | 1995 (Be Inc.) | 멀티미디어 특화 *media OS*. 오픈소스 후속 [Haiku](https://en.wikipedia.org/wiki/Haiku_(operating_system))로 명맥 ([BeOS](https://en.wikipedia.org/wiki/BeOS)) |

> **소비자(9x) → NT 통합:** 이 시기 Microsoft는 DOS 기반 9x 라인(95/98/Me)과 별도 설계의
> NT 라인을 함께 운영하다, **2001년 Windows XP에서 둘을 통합**했다. 자세한 계보는
> [Windows (DOS→NT)](/profiles/windows) 참조.

## 제6세대 (2007~현재) — 모바일·클라우드·AI

스마트폰 등장 + 가상화·컨테이너 + ML/AI 가속.

| OS | 출시 | 특징 |
|---|---|---|
| **iOS** | 2007 (Apple) | Darwin 기반 (macOS와 커널 공유) |
| **Android** | 2008 (Google) | Linux 커널 기반 |
| **Chrome OS** | 2011 (Google) | Linux 기반 웹 중심 OS |
| **컨테이너 OS** | 2014~ | Container Linux, RancherOS, Bottlerocket — 컨테이너 호스팅 전용 |
| **HarmonyOS** | 2019 (Huawei) | 멀티 디바이스 분산 OS |
| **AI-네이티브 OS** | 2024~ (실험적) | Rutgers의 AIOS, [GeulOS](/geulos) 등 — *LLM/객체 모델*을 OS 1급 기능으로 |

## 시대별 핵심 변화 정리

- **1세대 → 2세대**: 사람이 직접 → *자동화*된 배치 처리.
- **2세대 → 3세대**: 한 번 한 프로그램 → *동시* 여러 프로그램.
- **3세대 → 4세대**: 큰 메인프레임 → 한 사람의 PC.
- **4세대 → 5세대**: 독립 PC → 네트워크 연결.
- **5세대 → 6세대**: 책상 위 PC → 손바닥 위 모바일 + 구름 위 클라우드.
- **6세대 → ?**: AI가 OS 1급 시민이 되는 시대 — [GeulOS 사례](/geulos).
